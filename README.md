## Setup
- Add `nuxt-feathers-vuex` dependency using yarn or npm to your project
- Add `nuxt-feathers-vuex` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-feathers-vuex',

    // With options
    ['nuxt-feathers-vuex', { /* module options */ }],
 ]
}
```
